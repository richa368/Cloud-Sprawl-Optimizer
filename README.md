
# Cloud Sprawl Optimization using Zombie VM Detection.



## Overview 

Cloud environments often suffer from resource wastage due to inactive or underutilized virtual machines (VMs), commonly called zombie VMs. These unused resources contribute to cloud sprawl, increasing costs and reducing system efficiency.

This project implements a machine learning-based approach to identify and classify such zombie VMs, helping optimize cloud resource usage.

## Objective -

* Detect inactive or low-usage virtual machines.
* Reduce unnecessary cloud resource consumption.
* Improve efficiency and cost management.

## Features

* Data preprocessing pipeline for cleaning and preparing input data.
* Machine learning model for classification.
* Detection of zombie (inactive) servers.
* Integrated system combining all components.


## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## Conclusion

This project demonstrates how machine learning can be applied to optimize cloud infrastructure by identifying and eliminating unused resources, improving both performance and cost efficiency.
